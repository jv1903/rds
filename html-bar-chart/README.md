# HTML Bar Chart Project

This project creates a simple bar chart using HTML, CSS, and JavaScript. The chart displays three categories: "EM PROJETO," "EM OBRA," and "CONCLUÍDO."

## Project Structure

```
html-bar-chart
├── index.html       # Main HTML document
├── css
│   └── styles.css   # Styles for the web page and chart
├── js
│   └── chart.js     # JavaScript code to generate the chart
└── README.md        # Project documentation
```

## Getting Started

To run this project, follow these steps:

1. **Clone the repository** or download the project files to your local machine.

2. **Open the `index.html` file** in your web browser. This file sets up the structure of the web page and includes the necessary CSS and JavaScript files.

3. **Ensure you have an internet connection** if you are using a CDN for Chart.js in the `index.html` file.

## Dependencies

This project uses Chart.js for rendering the bar chart. You can include it via a CDN link in the `index.html` file or install it via npm if you prefer.

## Customization

You can customize the chart by modifying the data in the `js/chart.js` file. Adjust the labels and values as needed to reflect your data.
